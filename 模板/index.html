    # 字段说明

    • 链接： 必填，资源的URL地址
    • 说明： 必填，简要描述资源的用途和特点
    • 使用场景： 选填，说明在什么情况下使用这个资源
    • 账号/权限： 选填，如果需要登录或有权限要求，说明如何获取
    • 备注： 选填，其他重要信息

    # 格式规范

    1. 一级标题 #： 大分类名称（如：开发工具、API文档等）
    2. 二级标题 ##： 小分类名称（如：代码管理、数据库管理等）
    3. 三级标题 ###： 具体资源名称
    4. 列表： 使用 - 开头的列表项
